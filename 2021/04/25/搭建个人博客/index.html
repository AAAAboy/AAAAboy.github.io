搭建自己的个人博客

1、安装 Node.js

下载地址：https://nodejs.org/zh-cn/

傻瓜式安装

安装完成后打开命令行，执行 node -v 和 npm -v 检查是否安装成功。

然后安装淘宝镜像源，执行 npm install -g cnpm --registry=https://registry.npm.taobao.org

2、安装hexo

执行命令 cnpm install -g hexo-cli

3、初始化

进入blog文件夹后执行 hexo init

访问http://localhost:4000即可看到你的博客。

安装并生成blog的目录应该是这样的：

写好的.md文件可以放入 blog\source_posts 文件夹下，就可以在网页上浏览了

4、部署博客到 github 上

4.1 创建一个仓库

4.2 在 blog 目录下安装 git 部署的插件

windows下执行命令：cnpm install hexo-deployer-git --save

linux下执行命令：cnpm install --save hexo-deployer-git

4.3 修改 blog 目录下的 _config.yml 文件

注意：配置中每个冒号后面都要加一个空格

4.4 执行 hexo d 回车将本地blog部署到github的仓库

访问 https://aaaaboy.github.io/ ，可以看到已经成功的将博客部署到github上了

5、更换主题

好看的一个主题：https://github.com/litten/hexo-theme-yilia

在 blog 目录下执行命令：git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

将喜欢的主题克隆到 blog/themes 目录下

修改配置文件 blog/_config.yml

在 blog 目录下执行：hexo clean hexo g hexo s

进入 http://localhost:4000 查看
